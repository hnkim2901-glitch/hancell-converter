# 변경 이력

## v1.0.0 (2024-05-07)

### 초기 릴리스

#### 주요 기능
- 드래그앤드롭 기반 파일/폴더 추가
- PDF, HTML, XLS, XLSX 다중 형식 변환
- 파일별 변환 상태 실시간 테이블 표시
- CSV 로그 자동 저장
- 실패한 파일 재시도 기능
- 프로세스 자동 재시작 (설정 가능)
- 오류 발생 시 스크린샷 자동 저장

#### 기술 스택
- PySide6: UI 프레임워크
- pywinauto: 한셀 자동화
- psutil: 프로세스 관리
- Pillow: 스크린샷 캡처

#### 구현된 모듈
- `converter/file_queue.py`: 파일 큐 관리
- `converter/hancell_worker.py`: 한셀 자동화 워커
- `converter/logger.py`: CSV 로깅
- `utils/process.py`: 프로세스 관리
- `ui/main_window.py`: 메인 UI
- `app.py`: 메인 애플리케이션

#### 배포
- PyInstaller를 사용한 exe 패키징 지원
- 설치/실행/빌드 배치 스크립트 제공
