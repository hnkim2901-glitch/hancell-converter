# 빠른 시작 가이드

## Windows에서 바로 시작하기

### 1단계: 설치

```batch
setup.bat
```

위 명령어를 실행하면 자동으로:
- Python 버전 확인
- 가상 환경 생성
- 필요한 라이브러리 설치

### 2단계: 실행

```batch
run.bat
```

또는

```batch
venv\Scripts\activate
python app.py
```

### 3단계: 사용

1. **파일 추가**: 한셀 파일(.cell, .nxl, .xls, .xlsx)을 드래그앤드롭
2. **형식 선택**: PDF, HTML, XLS, XLSX 중 원하는 형식 체크
3. **출력 폴더 지정**: 변환된 파일이 저장될 위치 선택
4. **변환 시작**: "변환 시작" 버튼 클릭

## exe 파일로 사용하기

### 빌드

```batch
build.bat
```

### 실행

```
dist\한셀변환기.exe
```

더블클릭으로 바로 실행 가능!

## 문제 해결

### 한셀을 찾을 수 없다고 나오면?

`config.py` 파일을 열어서 `HANCELL_COMMON_PATHS`에 설치된 한셀 경로를 추가하세요.

```python
HANCELL_COMMON_PATHS = [
    r"C:\Your\Custom\Path\HCell.exe",
    ...
]
```

### 변환이 실패하면?

1. `logs/` 폴더에서 로그 확인
2. `screenshots/` 폴더에서 오류 화면 확인
3. 한셀 버전에 따라 `converter/hancell_worker.py`의 키 입력 순서 조정 필요

## 권장 설정

- **프로세스 재시작 간격**: 50개 (기본값)
  - 대량 파일 변환 시 안정성 향상
  - 메모리 누수 방지

- **기존 파일 스킵**: 체크 (기본값)
  - 중복 변환 방지
  - 시간 절약

- **하위 폴더 포함**: 필요에 따라 선택
  - 폴더 전체를 한 번에 처리할 때 유용

## 팁

1. **대량 파일 처리 전 테스트**: 먼저 1-2개 파일로 테스트해보세요
2. **백업**: 원본 파일은 항상 백업하세요
3. **실패한 파일 재시도**: 네트워크 드라이브나 느린 저장소에서는 타임아웃이 발생할 수 있으니 재시도 기능을 활용하세요
4. **로그 확인**: 변환 후 로그를 확인하여 문제 파일을 파악하세요
